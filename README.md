# 💼 AI Career Personality Scanner

An AI-powered desktop application that analyzes personality traits and predicts suitable career paths using Machine Learning.

Built with Python, CustomTkinter, Pandas, NumPy, Matplotlib, and Scikit-learn.

---

# 🚀 Features

✨ Modern futuristic dark UI  
✨ AI-powered career prediction  
✨ Personality trait analysis  
✨ Radar chart visualization  
✨ Interactive assessment system  
✨ Progress tracking sidebar  
✨ Dynamic career aura system  
✨ Machine Learning using Decision Tree Classifier  
✨ Restart assessment functionality  

---

# ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-career-personality-scanner.git
```

Move into project folder:

```bash
cd ai-career-personality-scanner
```

Install dependencies:

```bash
pip install customtkinter matplotlib pandas numpy scikit-learn
```

Run the application:

```bash
python main.py
```
